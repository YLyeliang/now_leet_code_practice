# 链表知识整理
链表与数组的不同之处在于：数组为顺序存储，每次插入和删除时需要移动后面的所有元素。链表为链式存储，通过不连续的方式，自适应内存大小，
并使用指针，当前的结点指向下一个结点的内存地址，来形成链表。

类型：单链表、双向链表、循环链表

```c++
typedef struct Node{
    int data;
    struct Node* next;
}Node,*LinkedList;
```
链表类问题在面试中主要考察的是面试人员是否对以下边际情况进行了考虑和解决:
- 取消对空指针的引用，通常以尾部指针为目标
- 当给定的头节点为空的时候
- 当链表中有重复数的时候
